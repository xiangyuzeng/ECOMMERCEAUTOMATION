# Wiki Screenshots

This directory contains screenshots referenced in 使用指南.md.

## How to capture screenshots

The wiki references the following images. To populate them:

1. Open the dashboard at http://localhost:3000
2. Take screenshots of each tab and save with the filenames below
3. For AdsPower screenshots, open AdsPower and capture the relevant screens
